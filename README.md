# Note de Frais - CSE Younup

Application de gestion de notes de frais pour les élus du CSE Younup.

## 🚀 Accès à l'application

L'application est accessible en ligne : [https://votre-username.github.io/note-frais](https://votre-username.github.io/note-frais)

## ✨ Fonctionnalités

- **Informations de l'élu** : Nom, prénom, adresse, email, téléphone
- **Types de frais** :
  - 🏨 Hôtel
  - 🅿️ Parking
  - 🍽️ Repas (plafonné à 25€)
  - 🚆 Transport
  - 📦 Divers
  - 🚗 Frais kilométriques (calcul automatique selon barème fiscal)
- **Calcul automatique** des montants HT, TVA et TTC
- **Génération de PDF** avec logo CSE et attestation sur l'honneur
- **Sauvegarde locale** des demandes en cours
- **Design responsive** (mobile-first)

## 📱 Compatibilité

L'application fonctionne sur :
- 💻 Ordinateur (Chrome, Firefox, Edge, Safari)
- 📱 Smartphone (iOS, Android)
- 📲 Tablette

## 🛠️ Développement local

Pour lancer l'application en local :

```bash
# Installer les dépendances
npm install

# Lancer le serveur de développement
npm start
```

Ou simplement ouvrir `index.html` dans un navigateur.
